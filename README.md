âœ… Stack Memory

Used for primitive data types

Number

String

Boolean

Null

Undefined

BigInt

Symbol

Key Characteristics:

Stores a copy of the value

Fast access

Fixed memory size

Changes do not affect the original variable

ğŸ“Œ Example: When assigning one primitive variable to another, a copy is created.

âœ… Heap Memory

Used for non-primitive (reference) data types

Objects

Arrays

Functions

Key Characteristics:

Stores data as a reference

Multiple variables can point to the same memory

Changes reflect in all references

ğŸ“Œ Example: When copying an object, both variables refer to the same memory location.

âœ… Stack vs Heap (Core Difference)

Fixed size	Dynamic size
